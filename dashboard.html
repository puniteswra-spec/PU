🔑 Re-authenticate GitHub

📋 Step-by-step: generate a new token (click to collapse)
  1. Click github.com/settings/tokens in a new tab
  2. Click Generate new token → choose Generate new token (classic) (recommended)
  3. Note: PunMonitor
  4. Expiration: No expiration (so it doesn't break in a year)
  5. Scopes: check ☑ repo (Full control of private repositories) — REQUIRED
  6. Scroll down, click Generate token
  7. Copy the token immediately (GitHub won't show it again)
  8. Come back here and paste it below
⚠ Don't use fine-grained tokens for PunMonitor — they have stricter scope rules. Classic tokens with repo scope work reliably.
Token should start with ghp_ (classic) or github_pat_ (fine-grained)
